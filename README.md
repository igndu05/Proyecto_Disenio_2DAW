# 🎵 Retrogroove

**Retrogroove** es una página web que funciona como un **e-commerce de discos de vinilo online**.  
Este proyecto ha sido desarrollado como **proyecto final de la asignatura de Diseño Web** en **2º de Desarrollo de Aplicaciones Web (DAW)**.

El objetivo del proyecto es simular una tienda online moderna dedicada a la venta de vinilos, permitiendo a los usuarios explorar productos, ver detalles, iniciar sesión y realizar compras a través de una pasarela de pago.

---

## 📌 Características principales

- 🛒 Catálogo de discos de vinilo
- 🔎 Vista detallada de producto
- 👤 Sistema de login de usuario
- 🧾 Carrito de compra
- 💳 Simulación de pasarela de pago
- 📱 Diseño web responsive
- 🎨 Interfaz centrada en la experiencia del usuario

---

## 📄 Páginas principales

El sitio web incluye varias secciones clave:

- **Home** – Página principal con destacados.
- **Login / Registro** – Acceso para usuarios.
- **Catálogo de productos** – Listado de discos disponibles.
- **Vista de producto** – Información detallada de cada vinilo.
- **Carrito de compra** – Gestión de productos seleccionados.
- **Pasarela de pago** – Proceso de compra final.
